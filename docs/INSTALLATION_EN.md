## Installation Steps

### Step 1: Download the Extension

1. Go to the [GitHub Releases page](https://github.com/omar173ahmed/tazkiyat/releases)
2. Find the latest release (v1.0.0)
3. Click on **`tazkiyat-extension-v1.0.0.zip`** to download
4. Save the file to your computer (e.g., Downloads folder)

### Step 2: Extract the ZIP File

1. Locate the downloaded ZIP file
2. **Right-click** on the file
3. Select **"Extract All..."** (Windows) or **"Open"** (Mac)
4. Choose a location to extract (e.g., Desktop or Documents)
5. Click **"Extract"** or **"Open"**
6. You should now have a folder named `tazkiyat-extension-v1.0.0`

### Step 3: Open Chrome Extensions Page

Choose one of these methods:

**Method A:** Using Chrome Menu
1. Open Google Chrome
2. Click the **three dots** (â‹®) in the top-right corner
3. Go to **More tools** â†’ **Extensions**

**Method B:** Using Address Bar
1. Open Google Chrome
2. Type `chrome://extensions/` in the address bar
3. Press **Enter**

### Step 4: Enable Developer Mode

1. On the Extensions page, look at the **top-right corner**
2. Find the **"Developer mode"** toggle switch
3. Click the toggle to **turn it ON** (it should turn blue)

> **Note:** This is required to install extensions manually. It's completely safe.

### Step 5: Load the Extension

1. Click the **"Load unpacked"** button (appears after enabling Developer mode)
2. A file picker window will open
3. Navigate to where you extracted the ZIP file
4. **Important:** Select the **`extension`** folder inside the extracted folder
   - Path should be: `.../tazkiyat-extension-v1.0.0/extension/`
5. Click **"Select Folder"** (or "Open" on Mac)

**Solutions:**
- Check your internet connection
- Verify the server URL in Settings is correct
- Try refreshing the page and retry

### Extension disappeared after restarting Chrome
**Solution:**
1. Go back to `chrome://extensions/`
2. Make sure Developer mode is still ON
3. Click the **reload icon** (ğŸ”„) on the ØªØ²ÙƒÙŠØ§Øª extension card
4. If it's not there, repeat installation steps

### "Authentication required" when sharing
**Solution:**
1. Click the extension icon
2. Logout and login again
3. Try sharing again

### Extension shows "This extension may be corrupted"
**Solution:**
1. Go to `chrome://extensions/`
2. Remove the extension
3. Download the ZIP file again
4. Repeat installation steps

## Updating the Extension

When a new version is released:

1. Download the new version ZIP file
2. Go to `chrome://extensions/`
3. Click **"Remove"** on the old version
4. Follow installation steps again with the new version
