  # Privacy Policy for Tazkiyat Extension

  Last updated: 11/01/2026

  ## Data Collection
  This extension does not collect, store, or transmit any personal data to third parties.

  ## Permissions Usage
  - **activeTab**: Used only to get the current tab's URL and title when you click the extension icon
  - **storage**: Used only to store your self-hosted server URL preference
  - **Host permissions**: Used only to communicate with your self-hosted Tazkiyat server

  ## Data Storage
  The only data stored is your custom API server URL, which is stored locally on your device using Chrome's storage API.

  ## Third-Party Access
  No data is shared with third parties. All communication is between your browser and your self-hosted Tazkiyat server.
